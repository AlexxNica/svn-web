GNOME has moved to Git

GNOME has changed to using Git for version control. Current GNOME sources can be found on git.gnome.org. All content on this site is obsolete.

Old Content

You can view GNOME Subversion interactively using the viewvc interface.

There is also some more detailed information for developers on what Subversion is and how to use it.

NOTE: If you have an existing CVS account, it will work with the Subversion service.

If you need a Subversion development account, please see the new accounts information page. For Frequently Asked Questions, please look at Subversion FAQ on the GNOME Wiki.

Ununsed SVN repositories may be moved to the SVN archive. These archived SVN repositories are available via: http://svn-archive.gnome.org/.
