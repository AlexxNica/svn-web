GNOME Subversion source code repository

You can view GNOME Subversion interactively using the viewcvs interface.

There is also some more detailed information for developers on what Subversion is and how to use it.

NOTE: If you have an existing CVS account, it will work with the Subversion service.

If you need a Subversion development account, please see the new accounts information page. For Frequently Asked Questions, please look at SubversionFAQ on the GNOME Wiki.
