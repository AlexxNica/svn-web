GNOME Subversion source code repository

IMPORTANT NOTE! Currently, the canonical source code revision control system for the GNOME project is CVS. Various alternatives to CVS are being discussed and evaluted. This is just an experimental Subversion service to assist us in these evaluations. Feel free to use it, but don't expect it to be up-to-date or accurate, or to work all the time. If the decision is made to switch to Subversion, this may become a live service. Please report any issues to the GNOME infrastructure mailing list.

For more information, see the notes on the work in progress.
